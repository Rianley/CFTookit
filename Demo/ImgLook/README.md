ImgLook--图片查看器
=====================

###### 作者:No.18
###### E-mail:cforth@cfxyz.com

一个简易图片查看器，使用Python3实现。

## 安装需要

* Python
    * Python 3.6
    * pycryptodome 3.4.6+
    * chardet 3.0.4
    * Pillow 5.0.0
* Windows(32位/64位)
    * Windows XP及以上
* Linux
    * python3-tk
    
## 使用方法

```bash
python3 ImgLook.py
```
支持常见图片格式（包括GIF动图）的显示，并且支持读取使用CFCrypto加密过的图片。
