MP3Player--音乐播放器
=====================

###### 作者:No.18
###### E-mail:cforth@cfxyz.com

一个简易音乐播放器，使用Python3实现。

## 安装需要

* Python
    * Python 3.6
    * pygame 1.9.3
    * chardet 3.0.4
* Windows(32位/64位)
    * Windows XP及以上
* Linux
    * python3-tk
    
## 使用方法

```bash
python3 MP3Player.py
```
实现了列表播放、循环播放、随机播放等一些MP3音乐播放器的功能。
